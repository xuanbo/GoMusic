## 2 运行

* 将本项目clone到GOPATH下
* 安装mysql驱动
  ```
  go get github.com/go-sql-driver/mysql
  ```
* 安装orm框架xorm
  ```
  go get github.com/go-xorm/xorm
  ```
* 安装web框架echo
  ```
  go get github.com/labstack/echo
  ```
* 启动 `go run main.go`
